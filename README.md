# CleanTheBeach
This project is aim to create a web site to collect post in facebook which clean the beach.
User make group and compete with each other to know who CleanTheBeach most.

## User 
User can login the web site with facebook account, and import their post to CleanTheBeach.
CleanTheBeach will parse the hashtag about beach cleaning and add score to their group
